# Case Café: Predição da Qualidade Final do Lote

Este repositório apresenta o **Case do Café**, um projeto educacional desenvolvido para o **treinamento de estagiários de Ciência de Dados na IBM**.

## Contexto do Negócio

O cliente do case, uma **fábrica de café** enfrentava dificuldades em compreender **quais fatores mais impactavam a qualidade final dos lotes de café** e **como prever esse nível de qualidade de forma consistente**.

A ausência dessa previsibilidade dificultava:
- O controle do processo produtivo  
- A otimização de decisões operacionais  
- A garantia de consistência na qualidade do produto final  

Diante disso, o desafio proposto foi utilizar dados do processo produtivo para **entender os determinantes da qualidade** e **prever o score final do café**.

## Objetivo do Case

- Explorar e compreender os dados disponíveis  
- Identificar os principais fatores que influenciam a qualidade do café  
- Desenvolver um **modelo preditivo** capaz de estimar o **score final de qualidade do lote**  
- Apoiar melhorias no controle do processo produtivo e na tomada de decisão  

## Estrutura do Case

O desenvolvimento do projeto foi dividido em **três etapas principais**, seguindo uma abordagem analítica completa:

### 1️. Análise Descritiva
- Exploração inicial dos dados  
- Entendimento das variáveis e da distribuição dos atributos  
- Identificação de padrões gerais e possíveis inconsistências  

### 2️. Análise Diagnóstica
- Investigação das relações entre variáveis  
- Identificação dos fatores que mais influenciam a qualidade do café  
- Análises direcionadas para explicar comportamentos observados  

### 3. Análise Preditiva
- Construção de modelos para prever o **score de qualidade do café**  
- Avaliação de desempenho dos modelos  
- Geração de insights para aplicação prática no processo produtivo  

## Contexto Educacional

Este case foi **desenvolvido por gerentes de Ciência de Dados da IBM**, que também são os **criadores da Academia de Dados**: programa de formação do qual participei e no qual este projeto foi realizado.

A Academia de Dados tem como foco o desenvolvimento prático de habilidades em Ciência de Dados, conectando conceitos técnicos a problemas reais de negócio.
